# Hospital Resource Utilization & Patient Outcomes Dashboard

## Problem Statement
Hospitals face challenges in efficiently managing beds, doctors, and patient flow across departments. This project provides an interactive dashboard to analyze hospital resource utilization and patient outcomes.

## Solution Overview
This dashboard visualizes key hospital metrics such as:
- Bed occupancy rates
- Patient admissions and outcomes
- Doctor utilization
- Average length of stay

It helps hospital administrators identify bottlenecks and improve operational efficiency.

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run the app:
   streamlit run app.py

## Outcome
The dashboard enables data-driven decision-making to optimize hospital resources and improve patient care.
